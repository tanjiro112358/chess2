CHESS PIECE SPRITES
==================

Place PNG files in this folder with the following names:

WHITE PIECES:
- white_pawn.png
- white_rook.png
- white_knight.png
- white_bishop.png
- white_queen.png
- white_king.png

BLACK PIECES:
- black_pawn.png
- black_rook.png
- black_knight.png
- black_bishop.png
- black_queen.png
- black_king.png

SPECIAL PIECES:
- clown.png

The game will automatically use PNG files if available,
otherwise it will use generated sprites.

Recommended size: 50x50 pixels or larger
Format: PNG with transparency
